# Space Guardian Remake

**Space Guardian Remake**는 과거 Unity로 개발되었던 슈팅 게임 `SpaceTyphoon`을 웹 기반(Phaser 3)으로 새롭게 이식한 프로젝트입니다.

## 🎮 게임 소개
지구(Earth)를 조작하여 우주에서 날아오는 위협(운석, 적 우주선)으로부터 살아남으세요!
열쇠를 모아 강력한 가디언을 소환하고, 거대한 보스 몬스터에 맞서 싸워야 합니다.

## ✨ 주요 기능
- **웹 기반 플레이**: 별도의 설치 없이 브라우저에서 바로 실행 가능합니다.
- **향상된 그래픽**: 레거시 프로젝트의 에셋을 고화질로 적용했습니다.
- **인트로 및 연출**: 게임 시작 화면, 카운트다운, 보스 등장 경고 등 몰입감을 높이는 연출이 추가되었습니다.
- **다양한 적과 보스**:
  - 일반 운석: 무작위로 떨어지는 기본 장애물
  - **보스 (Space Typhoon)**: 500점 달성 시 등장하는 거대 보스 (3방향 미사일 공격)
- **가디언 소환 시스템**: 열쇠를 소비하여 자동으로 적을 공격하는 아군 유닛 소환
- **사운드 효과**: 타격감 있는 폭발음과 효과음 적용 (Base64 임베딩)

## 🕹 조작 방법
- **이동**: `←`, `→` 방향키
- **가디언 소환**: `SPACE` 바 (열쇠 5개 필요, 쿨타임 1초)
- **일시정지**: `ESC` 또는 `P` 키

## 🛠 실행 방법
이 게임은 모든 리소스(이미지, 사운드)가 하나의 HTML 파일 안에 포함되어 있습니다.
1. `index.html` 파일을 크롬(Chrome) 등의 최신 웹 브라우저로 엽니다.
2. 화면을 클릭하여 게임을 시작합니다.

## 📦 기술적 특징
- **Phaser 3 Engine**: 가볍고 강력한 웹 게임 엔진 사용
- **Asset Injection**: 로컬 보안 정책(CORS) 문제를 해결하기 위해, Python 스크립트를 사용하여 모든 이미지와 사운드 리소스를 **Base64** 형태로 HTML 내부에 주입했습니다.
